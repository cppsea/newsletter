🐳 Who are we?

We're the "largest" software engineering organization on campus! We want to promote a safe place for all backgrounds to be excited and curious about software engineering!

We want to help you put software lifecycles methodologies, version control, documentation, data structures & algorithms, and the latest demand technologies used in the industry into practice.

So if you’re interested in software engineering or even coding in general? Check us out!
